# Naqsh

Naqsh is a full-stack e-commerce storefront for artificial/fashion jewellery, clutches & evening bags, and jewelry organizers, built for the Pakistani market.

## Stack

- **Backend**: Node.js, Express, MongoDB (Mongoose), JWT auth, Cloudinary image hosting, Nodemailer.
- **Frontend**: React 18, Vite, Redux Toolkit, React Router v7, Tailwind CSS.
- **Payments**: Cash on Delivery + PayFast (Pakistan) hosted checkout.

## Getting started

```bash
cd backend && npm install && npm run seed && npm run dev
cd frontend && npm install && npm run dev
```

Copy `.env` values before running — see `backend/.env` and `frontend/.env`.
